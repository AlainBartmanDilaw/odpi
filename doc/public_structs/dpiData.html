dpiData¶

This structure is used for passing data to and from the database for variables and for manipulating object attributes and collection values.

int dpiData.isNull¶

Specifies if the value refers to a null value (1) or not (0).

union dpiData.value¶

Specifies the value that is being passed or received.

int dpiData.value.asBoolean¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_BOOLEAN. The value should be either 1 (true) or 0 (false).

int dpiData.value.asInt64¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_INT64.

int dpiData.value.asUint64¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_UINT64.

int dpiData.value.asFloat¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_FLOAT.

int dpiData.value.asDouble¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_DOUBLE.

int dpiData.value.asBytes¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_BYTES. This is a structure of type dpiBytes.

int dpiData.value.asTimestamp¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_TIMESTAMP. This is a structure of type dpiTimestamp.

int dpiData.value.asIntervalDS¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_INTERVAL_DS. This is a structure of type dpiIntervalDS.

int dpiData.value.asIntervalYM¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_INTERVAL_YM. This is a structure of type dpiIntervalYM.

int dpiData.value.asLOB¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_LOB. This is a reference to a LOB (large object) which can be used for reading and writing the data that belongs to it.

int dpiData.value.asObject¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_OBJECT. This is a reference to an object which can be used for reading and writing its attributes or element values.

int dpiData.value.asStmt¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_STMT. This is a reference to a statement which can be used to query data from the database.

int dpiData.value.asRowid¶

Value that is used when dpiData.isNull is 0 and the native type that is being used is DPI_NATIVE_TYPE_ROWID. This is a reference to a rowid which is used to uniquely identify a row in a table in the database.
