dpiContext¶

This structure represents the context in which all activity in the library takes place. The implementation for this type is found in dpiContext.c. A context is created by calling the function dpiContext_create() and is destroyed by calling the function dpiContext_destroy().

uint32_t dpiContext.checkInt¶

Specifies a check integer which is used to verify that the handle passed by the application or driver is valid. It is set to 0 when the context is destroyed and to the value DPI_CONTEXT_CHECK_INT when the context is created.

unsigned int dpiContext.majorVersion¶

Specifies the major version passed when the context was created.

unsigned int dpiContext.minorVersion¶

Specifies the minor version passed when the context was created.
