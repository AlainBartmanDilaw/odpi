ODPI-C Private Structure dpiSubscr¶

This private structure represents subscriptions to events such as continuous query notification and object change notification and is available by handle to a calling application or driver. The implementation for this type is found in dpiSubscr.c. Subscriptions are created by calling the function dpiConn_newSubscription() and are destroyed by calling the function dpiSubscr_close() or by releasing the last reference when calling the function dpiSubscr_release(). All of the attributes of the structure dpiBaseType are included in this structure in addition to the ones specific to this structure described below.

dpiConn *dpiSubscr.conn¶

Specifies a pointer to the dpiConn structure which was used to create this structure.

void *dpiSubscr.handle¶

Specifies the OCI subscription handle.

dpiSubscrQOS *dpiSubscr.qos¶

Specifies the quality of service flags used by the subscription. This will be one or more of the values from the enumeration dpiSubscrQOS, OR’ed together.

dpiSubscrCallback dpiSubscr.callback¶

Specifies the callback that will be called when an event is propagated for this subscription. For more information see dpiSubscrCreateParams.callback.

void *dpiSubscr.callbackContext¶

Specifies the user-defined callback context pointer that is passed to the callback when it is executed. For more information see dpiSubscrCreateParams.callbackContext.
