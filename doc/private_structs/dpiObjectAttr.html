ODPI-C Private Structure dpiObjectAttr¶

This private structure represents attributes of the types created by the SQL command CREATE OR REPLACE TYPE and is available by handle to a calling application or driver. The implementation for this type is found in dpiObjectAttr.c. Attributes are created by a call to the function dpiObjectType_getAttributes() and are destroyed when the last reference is released by calling the function dpiObjectAttr_release(). All of the attributes of the structure dpiBaseType are included in this structure in addition to the ones specific to this structure described below.

dpiObjectType *dpiObjectAttr.belongsToType¶

Specifies a pointer to the dpiObjectType structure to which this attribute belongs.

const char *dpiObjectAttr.name¶

Specifies the name of the attribute, as a byte string in the encoding used for CHAR data.

uint32_t dpiObjectAttr.nameLength¶

Specifies the length of the dpiObjectAttr.name member, in bytes.

dpiDataTypeInfo dpiObjectAttr.typeInfo¶

Specifies the type of data stored in the attribute. This is a structure of type dpiDataTypeInfo.
