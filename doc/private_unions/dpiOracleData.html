ODPI-C Private Union dpiOracleData¶

This private union is used to avoid casts. The data referenced here is the data that is actually bound to or fetched from statements and represents the array of values that have been bound or will be fetched. It is used by the structure dpiVar and it is also used for getting data into and out of Oracle object instances.

void *dpiOracleData.asRaw¶

Specifies the pointer to the buffer allocated for the variable. The buffer allocated for the variable is a single contiguous piece of memory sectioned into array elements specific to the type of data being bound or fetched.

char *dpiOracleData.asBytes¶

Specifies the buffer as a series of byte buffers of the length corresponding to the variable member dpiVar.sizeInBytes.

float *dpiOracleData.asFloat¶

Specifies an array of floats.

double *dpiOracleData.asDouble¶

Specifies an array of doubles.

int64_t *dpiOracleData.asInt64¶

Specifies an array of 64-bit integers.

dpiOciNumber *dpiOracleData.asNumber¶

Specifies an array of OCINumber structures.

dpiOciDate *dpiOracleData.asDate¶

Specifies an array of OCIDate structures.

void **dpiOracleData.asTimestamp¶

Specifies an array of OCIDateTime handles.

void **dpiOracleData.asInterval¶

Specifies an array of OCIInterval handles.

void **dpiOracleData.asLobLocator¶

Specifies an array of OCILobLocator handles.

void **dpiOracleData.asString¶

Specifies an array of OCIString handles.

void **dpiOracleData.asStmt¶

Specifies an array of OCIStmt handles.

void **dpiOracleData.asRowid¶

Specifies an array of OCIRowid handles.

int *dpiOracleData.asBoolean¶

Specifies an array of booleans.

void **dpiOracleData.asObject¶

Specifies an array of object instances.

void **dpiOracleData.asCollection¶

Specifies an array of OCIColl handles.
