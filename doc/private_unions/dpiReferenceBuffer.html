ODPI-C Private Union dpiReferenceBuffer¶

This private union is used to avoid casts. It is used by the dpiVar structure to store references to LOBs, objects and statements that are bound to statements or fetched from the database.

void *dpiReferenceBuffer.asHandle¶

Specifies a generic handle pointer.

dpiObject *dpiReferenceBuffer.asObject¶

Specifies a pointer to a dpiObject structure.

dpiStmt *dpiReferenceBuffer.asStmt¶

Specifies a pointer to a dpiStmt structure.

dpiLob *dpiReferenceBuffer.asLOB¶

Specifies a pointer to a dpiLob structure.

dpiRowid *dpiReferenceBuffer.asRowid¶

Specifies a pointer to a dpiRowid structure.
