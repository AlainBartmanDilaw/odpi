ODPI-C Private Union dpiOracleDataBuffer¶

This private union is used to avoid casts. Unlike the data in the union dpiOracleData, which is intended to be placed on a memory location that is an array of buffers, this union itself contains the memory needed to supply the buffer that is being transferred to or from the Oracle database.

int64_t dpiOracleDataBuffer.asInt64¶

Specifies a 64-bit integer buffer.

uint64_t dpiOracleDataBuffer.asUint64¶

Specifies a 64-bit unsigned integer buffer.

float dpiOracleDataBuffer.asFloat¶

Specifies a floating point number buffer.

double dpiOracleDataBuffer.asDouble¶

Specifies a double floating point number buffer.

dpiOciNumber dpiOracleDataBuffer.asNumber¶

Specifies an OCINumber structure buffer.

dpiOciDate dpiOracleDataBuffer.asDate¶

Specifies an OCIDate structure buffer.

boolean dpiOracleDataBuffer.asBoolean¶

Specifies a boolean buffer.

void *dpiOracleDataBuffer.asString¶

Specifies an OCIString handle buffer.

void *dpiOracleDataBuffer.asTimestamp¶

Specifies an OCIDateTime handle buffer.

void *dpiOracleDataBuffer.asLobLocator¶

Specifies an OCILobLocator handle buffer.

void *dpiOracleDataBuffer.asRaw¶

Specifies a generic pointer which can be used to refer to the buffer as a whole.
